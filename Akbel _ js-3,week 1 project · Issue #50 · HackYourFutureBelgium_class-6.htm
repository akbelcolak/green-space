Dear Akbel,
I checked your GitHub Version 2 is different from master, definitely you have not push your master to remote repo. Probably you have done like this:

  1. You have finished version 2 and add changes, commit and push version 2
  2. You checkout master, then merge version2 to master.

Now you have to push your merged master to your repo again since you have merged in your laptp. You have to let GitHub know about this merge.
git push origin master
Then it should be ok. Good luck.
